# Вычисление расстояния Хаусдорфа между многоугольниками (Multi-Step алгоритм)

В этом проекте реализован алгоритм вычисления **расстояния Хаусдорфа между двумя многоугольниками** с использованием итеративной оптимизации на основе **субградиентов** и **последовательного сужения допустимой области**.

Алгоритм поддерживает как **выпуклые**, так и **невыпуклые** многоугольники.
## Архитектура
- `geometry.py` - базовые геометрические структуры и вычисления
- `solver.py` - реализация алгоритма оптимизации
- `main.py` - запуск обработки данных
- `data/` - входные данные и результаты
## Инструкция по установке
Создайте и активируйте виртуальное окружение:
```
python -m venv venv
venv\Scripts\activate
```
Установите необходимые зависимости.
```
pip install -r requirements.txt
```
## Использование 
1. Создайте папку `data` в корне проекта
2. Поместите туда наборы данных:
   - папки с многоугольниками
   - файл `metadata.csv`
3. Запустите:
```
python src\main.py
```
## Результат
После выполнения:
- файл `metadata.csv` будет обновлён
- появится новый столбец `hausDistMultiStep`
- для каждой пары многоугольников будет записано вычисленное расстояние Хаусдорфа
